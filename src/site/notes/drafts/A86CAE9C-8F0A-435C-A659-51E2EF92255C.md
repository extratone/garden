## Retrieve JSON data for all photos within a Snap.as account.

This shortcut uses the [Snap.as API](https://developers.snap.as/docs/api) to retrieve oneâ€™s entire Snap.as library in JSON format.  By default, a [Set Name](https://www.matthewcassinelli.com/actions/set-name/) action sets the name of the resulting file to `snapas.json` before prompting the user with a [Save File](https://www.matthewcassinelli.com/actions/save-file/) action.

Upon installation, you will be required to set your Write.as/Snap.as user token.
