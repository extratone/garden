## Description

Create a Markdown-formatted GitHub Gist for each new iOS contact.

Example: https://gist.github.com/extratone/0373594b7b16d5c5a617e2233a134044

Default template:

```
# Name
*Added `DateAdded`*
Organization, JobTitle

- [Phone](tel:PhoneNumber
- [Email](mailto:Email)

Address

Notes
```
