#### Configuration
| Field            | Content                                             |
|:----------------:|:---------------------------------------------------:|
| API URL          | https://snap.as/api/photos/upload                   |
| HTTP Method      | POST                                                |
| HTTP Headers     | Authorization: 00000000-0000-0000-0000-000000000000 |
| HTTP Body        | (Empty)                                             |
| Form File Name   | file                                                |
| JSON Result Path | data.url                                            |
